# “赓脉拓新・守正知行” 传习实践团网页
## 团队简介
“赓脉拓新・守正知行” 传习实践团是一支以青年学子为主体的调研团队，聚焦传统文化创新表达与传播，走进故宫、中国工艺美术馆开展双馆调研，探寻传统文化创造性转化与创新性发展路径，争做党的创新理论传播者、传统文化创新践行者
